[説明 / About]\
Unity上で動作するパチスロシミュレーターです。過去作で作成した"Nostalgia"のリメイク版を同梱しています。

### SampleGame Instructions (Nostalgia)

[言語設定 / Language]\
初回起動時に日本語か英語のいずれかから言語設定ができます。\
\
[操作方法 / How to play]\
遊び方、操作説明はゲーム内メニューからも確認できます。\
\
スペースキー:MAX BET\
1キー:1BET\
2キー:2BET\
上キー:MAX BET&リール始動\
\
左キー:左停止\
下キー:中停止\
右キー:右停止\
*リールは1分経過で自動停止します。\
\
Aキー:オートプレイ切り替え\
Mキー:メニュー開閉\
\
[セーブタイミングについて / Save timing]\
セーブは第三停止ボタンを離したタイミングでセーブされます。払い出しの途中であっても\
ロード時には払い出しがすでに終わった状態でゲームが開始します。\
\
[打ち止めについて / Spin limit]\
99999G到達で打ち止めになります。\
\
